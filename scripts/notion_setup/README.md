# Notion Setup

The LifeOS Notion parent page and six data sources already exist. Do not recreate them.

Use this folder only for future setup notes, migration notes, or validation scripts that inspect the existing Notion workspace.

Required access:

1. Create or use a Notion integration.
2. Share the existing LifeOS parent page with the integration.
3. Confirm the integration can read the six existing data sources documented in `NOTION_SCHEMA.md`.
4. Store the integration token in `.env` locally or in GitHub Actions secrets.
